Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Je hebt drie manieren en dat zijn:

  • External CSS
  • Internal CSS
  • Inline CSS

external css = met een external style sheet kan je de uiterlijk van een website veranderen door 1 css file. Elk html pagina moet een link element hebben dat refereert naar de css file. De file moet wel eindigen in .CSS

Internal CSS = Deze zit in de html pagina zelf en is alleen bestemd voor dit .hmtl bestand. Deze kan je vinden in de head section onder style

inline CSS = wordt gebruikt voor een enkel element. Dit gebruik je in een .html file. bijv

<"style="color:blue;text-align:center;">

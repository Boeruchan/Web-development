Casade

De regels van CSS is de laatste code is van toepassing
Bijvoorbeeld: je hebt in css 2x h1, eerste is rood, tweede is blauw.
In de body zal bij de h1 blauw kleuren want die is de laatste dat geschreven is.
De eerste h1 in rood is dan geannuleerd. Tenzij er een class wordt aangegeven aan de h1 dan zal de class de laatste geschreven code overrulen.

voorbeeld

h1 {
color: red;
}
h1 {
color: blue;
}
h1 { color: red; } h1 { color: blue; } <h1>This is my heading.</h1><br>

This is my heading.


Inheritance

Sommige CSS property values heeft een parent element en die zijn gekoppeld aan de child element.

body {
color: green;
}

span {
color: black;
}

<p>As the body has been set to have a color of blue this is inherited through the descendants.</p>
<p>We can change the color by targetting the element with a selector, such as this <span>span</span>.</p>

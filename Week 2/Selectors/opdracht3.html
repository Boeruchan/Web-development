Selectors & visuals rules

Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld;

  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.
  • Universal selector, type selector, class selector, id selector, attribute selector
  • descendant selectors: Bij descendent element-selectors wordt de stijl alleen aan een element toegevoegd, wanneer deze een nakomeling (descendent) is van één of meer andere elementen.
  • class selectors: matches elements based on the contents of their class attribute.

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?


H is een selector, dat richt zich op een html element die je wilt stylen.
color: is een property en blauw is de property kleur.
font-size: is property text grootte en 12px is de grootte van de tekst.

Hoe geef je 2 of meer parameters mee aan een functie? Geef een voorbeeld. 

      const a = 1;
      const b = 2;

      const answer = (a, b) => a + b;
      console.log(answer(a, 2));
      // answer: 3

Wat zijn rest parameters? Geef een voorbeeld.

    const nums = (first, ...rest) => rest;
    console.log(nums(1, 2, 3, 4));
    // answer: [ 2, 3, 4 ]

    const num = (first, second, ...rest) => rest;
    console.log(num(1, 2, 3, 4));
    // answer: [3, 4]

Wat is het verschil tussen rest parameters en expliciete parameters die je aangeeft bij een functie?

implicite (javascript convert uit zichzelf de datatype):


      const a = 1;
      const b = 2;
      const c = "2";

      const answer = (a, b) => a + b;
      console.log(answer(a, 2));
      // answer: 3

      const answer2 = (a, c) => a + c;
      console.log(answer2(a, c));
      // answer2 = 12
      (a got converted to a string)

      const answer3 = (a, c) => a * c;
      console.log(answer3(a, c));
      // anser3 = 2
      (a got converted to a number)
